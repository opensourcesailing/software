# 💻 Open Source Sailing Software

## Open-source software for marine technology

This repository contains software projects developed for Open Source Sailing.

The goal is to create open, reliable and accessible software solutions that connect marine systems, data and users.

---

# Software Philosophy

Open Source Sailing software should be:

⚓ Open  
Source code should be available and understandable.

🔗 Interoperable  
Systems should communicate using open standards.

🔧 Maintainable  
Software should be documented and easy to improve.

🤝 Community-driven  
Development should happen openly.
